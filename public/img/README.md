This folder contains image assets used by the Capability Matrix application.

Place production assets here and reference them using absolute paths from the app (for example: `/img/logo.svg`).

Suggested filenames:
- `logo.svg` — primary logo (light)
- `logo-dark.svg` — logo optimized for dark backgrounds
- `avatar-placeholder.svg` — generic avatar placeholder
- `favicon.ico` — optional site favicon

Best practices:
- Use SVG for logos and icons where possible (scales crisply and is theme-friendly).
- Optimize images for web (use tools like svgo or pngquant).
- Keep filenames lowercase and use `-` as separator.
